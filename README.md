## Resume Scraper & PDF Generator
This is a web app that scrapes data from static LinkedIn profile HTML files and generates a downloadable PDF resume.

## Tech Stack
- React
- Vite
- Puppeteer
- react-pdf
- Node.js

## How to Run It
bash
Copy
Edit
git clone https://github.com/DinoJazvin/resume-scraper.git
cd resume-scraper
npm install
npm start
